Tips之打造个人品牌

发表于 2019-03-07 | 分类于 ARTS | | 阅读次数:

原文链接在这里，不知道需不需要翻墙。

呐，上面的链接是打不开的，因为这篇文章是我自己对自己说的一些话。

我们在职场中，个人是乙方，单位是甲方。单位提供平台、资源、环境，我们拿钱干活，为公司输出自己的价值。所以，不存在弱势、强势的问题。就是一个议价问题嘛，和处对象一样，处的来就一直处下去，性格不合就好聚好散嘛。

所以，无论在哪家公司，为谁服务，我们都需要在工作中、在生活中注意提升自己，把自己当做品牌来经营。呐，什么样的品牌值得信赖呢？有能力当然是很重要的，但是我认为靠谱更重要。

一个靠谱的人是什么样的，就是领导将事情交给你很放心，不是说一定相信你的能力能摆平，而是说你会认认真真的对待，就算搞不定，也会想出一些方案应对。

大家在职场行走，无论是同事也好，还是领导也好，在同一家公司，大多数会融洽的相处，如果脱离公司这个平台呢，你有多少能力或者资源活下去？

当然了，如果你能力很强，现在的社会一定不会缺少机会的，但是如果你能力很强，你当然不会失业，但是请你对自己要求再高一点，我们不是说有个工作就会去做。我们还是要去做一些自己感兴趣，对社会有意义的事情。当然了，这只是个人的一些看法，也许会有些偏颇。

在工作中，我努力的目标就是上面说的靠谱，当别人有事的时候能够第一个想到你。

认认真真做事，不只是为了对得起公司付给我们的薪水，也是一份职责和担当，但是更多的是对自己这个品牌的负责。

一个人的技能可以提高，性格也可以刻意培养，靠谱这种能力也可以。

经营好我们自己这个品牌，无论在职场还是在生活中，相信会会更舒服或者自由一些。

写的比较乱，个人文笔功底也是差到爆，各位看官多担待，这个博客希望是我文笔提升的一个见证。

Tips之Why I Still Use Vim

发表于 2019-02-18 | 分类于 ARTS | | 阅读次数:

原文链接在这里，不知道需不需要翻墙。

新年第一周，又开始了一年的忙碌，首先祝大家身体健康，阖家欢乐！

第一周给大家分享的文章是关于vim的，当然关于使用什么编辑器，那是仁者见仁智者见智、是一件非常个性化的事情。

文章从打开文件占用的内存、打开文件的速度两方面做了对比，还分享了一个学习vim的在线网站和一本书。

具体内容就请大家移步原文品读了，下面说说我个人关于编辑器的一点点想法。

文章开题，其实已经表明了博主的态度，使用什么编辑器是十分个性化的事情，可以说每个人都有自己用着顺手的编辑器，甚至是基于个人的特定习惯个人定制化很多功能。

比如博主，个人开发时常用的编辑器也是vim，自己偶尔也写一些插件、风格化的文件等等，因为这些东西都是基于个人习惯定制的，而且一旦写好几乎很多年不用修改的，所以越用越顺手，越用越感觉其他编辑器无法满足自己的需求了。

但是，如果我们是一个大的团队协同开发一些大项目，vim就会稍显力不从心，很多文件管理需要团队规定，比如文件归类、配置选项等等。这种情况还是建议大家用一些正规军，方便项目管理。

以上就是关于vim的一些拙见，仅供探讨。
