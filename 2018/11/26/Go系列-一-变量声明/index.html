发表于 2018-11-26 | | 阅读次数:

golang 整体看起来像是like c风格，常用变量类型有

变量声明方式有以下三种：

  • 1、var 变量名 变量类型
    (可用作全局变量声明)
  • 2、var 变量名 = value，自行判定变量类型
  • 3、省略var 变量名:=value，自行判定变量类型，只能用于函数内部，不可用作全局变量声明
